# Suggested Repository Labels and Topics

These suggestions are meant to make work easier to filter across the Limes Labs organization. Apply them deliberately; there is no need to create labels that are not yet used.

## Shared labels

| Label | Purpose |
| --- | --- |
| `good first issue` | Small, well-scoped task for a new contributor |
| `help wanted` | Maintainers want outside contribution |
| `needs source` | Requires public sources before merge |
| `research` | Research, literature, methodology, or source review |
| `documentation` | README, guide, template, or public note |
| `policy` | Legal, policy, procurement, licensing, or governance work |
| `security` | Security-sensitive work or disclosure follow-up |
| `benchmark` | Evaluation task or benchmark methodology |
| `compute` | Compute access, infrastructure, GPU-hour estimates |
| `reproducibility` | Commands, configs, logs, and repeatable results |
| `question` | Needs maintainer clarification |
