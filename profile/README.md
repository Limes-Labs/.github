# Limes Labs

**We are not claiming capability yet. We are organizing capability.**

Limes Labs is an early European open AI capability project focused on open source software, open weight models, transparent evaluation, safety, and digital sovereignty.

- Website: [limeslabs.eu](https://limeslabs.eu)
- Join: [limeslabs.eu/join](https://limeslabs.eu/join)
- Open questions: [limeslabs.eu/open-questions](https://limeslabs.eu/open-questions)
- Contact: hello@limeslabs.eu

## Start here

New contributors should be able to find a useful first task in under five minutes.

1. Read [CONTRIBUTING.md](https://github.com/Limes-Labs/.github/blob/main/CONTRIBUTING.md) to choose the right repository and contribution path.
2. Check [ROADMAP.md](https://github.com/Limes-Labs/.github/blob/main/ROADMAP.md) for the current 30-day operating layer and the boundaries of what we are not doing yet.
3. Open a focused issue using one of the templates, or pick an issue labeled `good first issue`, `help wanted`, or `needs source`.

Useful roles include researchers, evaluation engineers, infrastructure engineers, ML engineers, technical writers, legal and policy contributors, institutional operators, and funders who can help turn questions into verifiable work.

## Current operating plan

1. Map European AI infrastructure and compute access paths.
2. Convert open questions into issues with sources, owners, and review criteria.
3. Build the contributor network through public issues, pull requests, and [limeslabs.eu/join](https://limeslabs.eu/join).
4. Ship small, reproducible experiments before making larger capability claims.
5. Publish what we contact, learn, succeed at, fail at, and still lack.
